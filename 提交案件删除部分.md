# 提交案件删除部分（I CERTIFY — TRANSMIT REPORT 点击后的交互步骤）

以下为点击「I CERTIFY — TRANSMIT REPORT」按钮后**已删除**的交互步骤内容，仅作记录保留。

---

## 1. 删除的代码位置

文件：`bot.py`，函数：`do_submit()`。

---

## 2. 删除的交互步骤一：Official Case Acceptance Notice（正式案件受理回执）

**原逻辑**：案件写入数据库并生成 PDF 缓存后，向用户发送长段「正式案件受理回执」文案，并附带操作按钮（Download PDF Summary、Check Case Status、PERMANENTLY CLOSE SESSION）。

    # ── Official Case Acceptance Notice ─────────────────────

**对应按钮**（`kb_after_submit(case_id)`）：
- 📂 Download PDF Summary

## 3. 删除的交互步骤二：SECURITY COMPLIANCE NOTICE（安全合规提示）

**原逻辑**：在发送回执后，再单独发送一条「SECURITY COMPLIANCE NOTICE」，并在 60 秒后自动删除该条消息。

**删除的代码：**

```python
    # ── Security Compliance Notice (sent separately for visibility) ──
    import asyncio
    security_msg = await bot.send_message(
        chat_id,
        "🛡️ <b>SECURITY COMPLIANCE NOTICE</b>\n"
        + "━"*28 + "\n\n"
        "To adhere to <b>Federal Data Protection Standards</b>, please:\n\n"
        "• <b>Manually delete</b> any sensitive messages or images\n"
        "  containing PII from this chat history within <b>5 minutes</b>.\n\n"
        "• The system has automatically purged all temporary\n"
        "  session data to protect your privacy.\n\n"
        "• Only your <b>Case ID</b> and <b>Hash Records</b> are retained\n"
        "  as query credentials.\n\n"
        f"🔖 <b>Your Case ID:</b> <code>{case_id}</code>\n\n"
        "<i>This notice will self-delete in 60 seconds.</i>",
        parse_mode="HTML",
    )

    # Auto-delete the security notice after 60 seconds
    async def _auto_delete_security_notice():
        await asyncio.sleep(60)
        try:
            await bot.delete_message(chat_id, security_msg.message_id)
        except Exception:
            pass

    asyncio.create_task(_auto_delete_security_notice())
```

## 4. 当前保留行为（修改后）

点击「I CERTIFY — TRANSMIT REPORT」后仅保留：

- 写入数据库（`db.create_case(record)`）
- PDF 缓存写入（`ctx.bot_data["pdf_cache"][case_id]`）
- 管理员通知（不变）
- 将「Processing」动画消息编辑为：**✅ Submission Complete — Case Registered**，并在同一条消息中展示 **Case ID**（便于用户记录），不再发送回执长文与后续按钮、不再发送 SECURITY COMPLIANCE NOTICE。

---

## 5. 说明

- 如需恢复上述交互步骤，可从此文件复制对应代码块回 `bot.py` 的 `do_submit()` 中。
- `kb_after_submit` 仍在 `keyboards.py` 中保留，可供 PDF 等回调（如 `pdf|{case_id}`）在其他入口使用。
